  • Browse to required directory (Normally public_html)
  • Upload the files inside Gute/app folder.

Color

How to change main color
  1. Go to path app/assets/sass/abstracts
  2. Open varible.scss file
  3. Find variable $colors and edit key yellow in it to color you want
  4. Finally, go to root folder, open terminal & run gulp build
